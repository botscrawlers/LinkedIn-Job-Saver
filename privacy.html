LinkedIn Job Saver

Política de Privacidad

Esta política de privacidad describe cómo la extensión LinkedIn Job Saver trata tus datos.

1. Recopilación de Datos

La extensión NO recopila, almacena ni comparte ningún dato personal. Toda la información que guarda (como el título del puesto, empresa, descripción de la oferta laboral y datos visibles del reclutador) se extrae directamente del sitio web de LinkedIn al momento de hacer clic en "Guardar esta postulación".

2. Almacenamiento de Datos

Los datos capturados se almacenan únicamente en tu propia cuenta de Google Drive, a través de una hoja de cálculo de Google Sheets vinculada por vos mismo/a.

En ningún momento la extensión envía, transmite ni almacena esta información en servidores externos del desarrollador o de terceros.

3. Permisos de la Extensión

  • Host: linkedin.com - necesario para acceder a las páginas de ofertas laborales.
  • Storage - se usa para guardar el ID de macro de tu hoja de Google Sheets localmente en tu navegador.
  • Scripting - permite ejecutar código en la pestaña activa para extraer la información.
  • Tabs - se usa para identificar la pestaña activa donde se encuentra la oferta laboral.

4. Seguridad

Como no se recolecta información personal ni se envían datos a servidores externos, el riesgo de exposición de datos es prácticamente nulo.

5. Cambios en esta política

Esta política puede ser actualizada en el futuro. Cualquier cambio será notificado a través del repositorio oficial de GitHub donde se aloja el proyecto.

6. Contacto

Si tenés preguntas o inquietudes sobre esta política de privacidad, podés contactarte a través del repositorio de GitHub del proyecto.
